Gemstone Price Prediction Project

A project to predict gemstone prices based on various features.

Project Details:

  • Project Name: Gemstone Price Prediction
  • Description: This project aims to predict the price of gemstones based on various features such as carat, cut, color, clarity, depth, and table.
  • Dataset: The dataset used in this project is taken from Kaggle.
  • Machine Learning Model: In this project, I used regression algorithms (e.g., linear regression, decision tree regression, or random forest regression) to build the prediction model.
  • Web Application: I created a flask application to allow users to input gemstone details and get price predictions.

Sample Features:

  • Carat (numerical): Weight of the gemstone in carats.
  • Depth (numerical): Total depth percentage (from the top to the bottom of the gemstone).
  • Table (numerical): Width of the top of the gemstone relative to the widest point.
  • Cut (categorical): Quality of the cut (e.g., Fair, Good, Very Good, Premium, Ideal).
  • Color (categorical): Color grade of the gemstone (e.g., D, E, F, G, H, I, J).
  • Clarity (categorical): Clarity grade of the gemstone (e.g., IF, VVS1, VVS2, VS1, VS2, SI1, SI2, I1).
